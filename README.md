# Blinky
This is my Blinking project for stasis event that is coming up.

<img width="855" height="641" alt="0" src="https://github.com/user-attachments/assets/b52bb426-4b5d-4179-bb8c-9d2255db0235" />

This project was made using one led resistor and a 3D-printed rocket ship I made on my printer. I wanted to have something that let me know if my PC was still on or not after t went to sleep, and this was the perfect way to solve it, With the artemis launch and project hail mary I took a lot of inspiration from those things.
